Search Techniques

Search Techniques is the process of searching an item from a list of element.


Types of Search Techniques

  • Uniformed Search(Blind search)

  • Informed Search(Heuristic Search)


Uniformed Search

They have no additional information about states beyond that provided in problem definition.

They have no preference as to which node it should explore first.

Procedure: Generate successors and distinguish a goal state from non-goal state.

Examples: Breadth First Search, Depth First Search, Depth Limited Search, etc

1, Breadth First Search

  • The Root node is expanded first, then the successor nodes are expanded, after which their successor nodes are expanded and so on.
  • It is also called Level-by-level traversing.
  • Downward motion procceds level by level until goal is reached

2, Depth First Search

  • The Root node is expanded first, then the first shallow successor node is expanded till the deepest level, then traverse back to the next shallow node and repeat until goal is reached.
  • It has a very modest memory requirements.

Informed Search

Strategies that know whether one `non-goal state` is more promising than another.

Employs heuristic(to discover/investigative) for its moves. Heuristic is a rule of thumb that probably leads to a solution.

Examples: Greedy Best First Search, A* Search, Hill Climbing

1, Greedy Best First Search

  • Greedy Best first search is a informed searching technique in which the algorithm seeks minimum cost to reach the goal.
  • It uses Heuristic function to decide which node to expand.
  • It is neither complete nor optimal.
  • It makes decisions based only on the information it has at any one step, without regard to the overall problem.
  • F(n) = H(n), where H(n) is the heuristic function result

2, A* Search

  • A* search is a informed searching technique in which the algorithm seeks minimum cost to reach the goal as well as cost to reach the node.
  • It is both complete nor optimal.
  • It is the widely known form of best first search.
  • F(n) = G(n) + H(n), where G(n) cost to reach the node H(n) is the heuristic function result

3, Hill Climbing

  • Hill climbing is a Depth first search with a heuristic measurement
  • It is an optimization techinque that is used to find Local Optimum solution to a computational problem.
  • It starts off with a solution that is very poor compared to the optimal solution and then iteratively improves from there.
  • It may not give the optimal solution as it might stop looking for further solutions thinking it has found the optimal solution when actually it is not.
