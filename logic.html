Logic

Logic is the process of thinking correctly. It is a act of correct reasoning.



There are different kinds of logic: classic logic, fuzzy logic, relevance logic, Propositional logic(Boolean logic), Predicate logic(First order predicate logic)

Here we learn only two:

Propositional Logic(Boolean Logic)

It is a simple form of logic where a symbol represents a fact.
Propositional symbol by themselves have no meaning for example:
A -> Ram is boy
B -> Today is hot
Boolean connectives are used to make complex sentences

Connective Description
¬ Not / Negation
⇒, → Implies / If...Then
⇔, ↔ Equivalent / If and only If 'Iff'
∨ OR
∧ AND
∀ For all (universal quantifier)
∃ there exists (existential quantifier)


A proposition is always either true or false, not both.

If for all interpretation of a formula is true then it is called a Valid Formula or Tautology
If for all interpretation of a formula is false then it is called a Inconsistent Formula or Contradiction



Predicate Logic (First Order predicate logic/calculus)

Predicate is an expression of one or more variable defined on some specific domain.
It has a rich set of vocabulory. One of main is that real world is made up of objects with individual properties and identites that distinguishes them from other objects.
Objects are bond together with a relationship.

Object = people, thing, place
Properties = red, sweet, tall
Relation = brother_of, side_to, has_color

Predicate is a special function with range {F,T}. Predicate has a name and argument list.
For example: P(X[1], X[2], ..., X[n]) here P is the name and X[1], X[2], .. X[n] are the arguments.
When all the arguments are initiated or given then the predicate becomes proposition that is either true or false.
father(gyanendra, y) = predicate
father(gyanendra, paras) = proposition
