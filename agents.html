Agents

An agent is anything that can be viewed as preceiving its environment through its sensor and acting upon that environment through actuators.

An agent consists of some sort of Sensors(cameras,sensors,rays,etc) that takes in the input from its environment and feed to the agent system after processing the agent then response with some sort of actions with helps of their Actuators(wheels,speakers,etc)


Types of Agents

  • Simple Reflex Agent
    • Simple If-then, ReactiveAct on given input. approach
    • Cannot learn from experience
    • If the percept is not in the agents database it cannot react properly
    • Example: simple mercury type thermostat/ heater, basic rover
  • Modal Based Reflex Agent
    • More Reactive than Simple Reflex agents
    • Keeps track of the experience it collects
    • Example: Rover with sample collection
  • Goal Based Agent
    • Has a ProactiveAct after making decision approach
    • It generally knows some knowledge on how the environment works. And it uses that knowledge to search and identify plan before acting.
    • It uses a Search and Plan pattern that helps it do be more proactive than reactive.
    • Example: Vaccum World Problem
  • Utility Based Agent
    • All the agents before had only one goal but it has multiple.
    • Uses a Utility Function to calculate the possibility of each plan to be successful.
    • It also looks into account the probability of success, how much resource is needed, what is the prority of the goal, how much time will it take, and many more.
    • Example: Auto Taxi

PAGE Description

PAGE stands for Percepts Actions Goals Environment.

Percepts :- Information accquired by the sensors of the agent.

Actions :- Operations performed by the agent using actuators.

Goals :- Desired outcome of the task with a measurable performance.

Environment :- Surroundings beyond the control of the agent.
