Expert System

An expert system is a computer that emulates the decision-making ability of a human expert.

It uses a knowledge base and a inference engineReasoning engine to come up with an answer to the problem given.


Working of an ES system

  • Step 1: Create a knowledge base for the system with help of human expert and other sources.
  • Step 2: Users uses query to ask to the ES system
  • Step 3: The ES system uses its inference engine to derive a conclusion using the knowledge base.
  • Step 4: User get their result back from the ES system.


Some fields where expert system are used

  • Classification: Identifying an object based on stated charactersitics
  • Diagonsis System: Infer malfunction and disease from observable data
  • Design: Configure a system according to specifications
  • Scheduling and planning: Develop or Modify a plan of action

Benefits of Expert System

  • Successful ES has very low error rates
  • ES make consistent recommendation
  • An ES can compile its part of the work much faster than human expert
  • An ES can be used in hazardeous environment
  • When used as a training vehicle, ES can be a great learning curve for novice

Limitations of Expert System

  • Limitations of the technology
  • Problems with knowledge acquistion
  • Operational domains as the principal area of ES application
  • Maintaing human experties in organization
